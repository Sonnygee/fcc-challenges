Introduction

In this course, we will be learning web development, absolutely free, from the very beginning. So I have been learning web development for weeks now through freeCodeCamp website and other learning references such as YouTube videos. Now, if you would just like to build a website for yourself or your business, this course is not for you. But, if you think web development is an on-demand career with high-paying checks, and if you believe that coding is cool, then this course is for you!

So, the first thing to do is to install some professionao web development tools. The good news is, they are all free! We all know that web developers love free stuff.

  1. Download and install Visual Studeio (VS) code on your web browser. Here's the link. This is the most popular text editor these days, and it's totally free!
  2. Once you instaled VS code on your computer/laptop, it is also helpful to install some extensions such as the LiveServer on the Marketplace.
  3. To start on a project, click on the File > Open a Folder, then select any folder on your computer to store all of the contents of your project in one place.
  4. Next, click on the 'New File' button and type 'index.html'. This is the standard name developers usually have if they are beginning coding. On the empty text editor, type '!' then 'Tab', and you would see the so-called "website basic template" as shown below. <!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Document</title> </head> <body> </body> </html>
  5. Finally, to see your progress real-time, you can use the live-server you installed on your VS code and run the code using your browser.

You can always save your progress on your computer or laptop using the keyboard command CMD + S

HTML

HTML is the standard markup language for creating Web pages.

What is HTML?

  • HTML stands for Hyper Text Markup Language
  • HTML is the standard markup language for creating Web pages
  • HTML describes the structure of a Web page
  • HTML consists of a series of elements
  • HTML elements tell the browser how to display the content
  • HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

What is an HTML element? An HTML element is defined by a start tag, some content, and an end tag: <tagname> Content goes here... <tagname> The HTML element is everything from the start tag to the end tag.

CSS

What is CSS?

  • CSS stands for Cascading Style Sheets
  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once
  • External stylesheets are stored in CSS files

Why use CSS? CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes. An example of a CSS code is as follows: body { background-color: lightblue; } h1 { color: white; text-align: center; } p { font-family: verdana; font-size: 20px; }

CSS Selectors

In CSS, selectors are patterns used to select the element(s) you want to style.

An example of such is as follows: .class > .intro > Selects all elements with class="intro"

CSS Pseudo-classes

A pseudo-class is used to define a special state of an element. For example, it can be used to:

  • Style an element when a user mouses over it
  • Style visited and unvisited links differently
  • Style an element when it gets focus

The syntax of pseudo-classes is shown below: selector:pseudo-class { property: value; }
